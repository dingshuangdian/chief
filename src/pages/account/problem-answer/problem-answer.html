疑问解答
  1. 我要如何开立会员卡？

    2. 在“会员开卡”中输入车主信息，选择需开立的电子会员卡即可，操作简单、快捷。

  2. 我如何对会员卡进行消费处理？

    3. 在“会员消费”中可根据会员卡、车牌号、手机号查询出该会员名下所有的会员卡，选择对应的会员卡进行消费即可。

    消费完成后，车主关注并绑定了微信公众号，可在微信中接收到一条消息通知的提醒。

    未关注微信公众号的车主，可直接受到消费通知的短信提醒。

  3. 我如何查看店面的预约情况？

    4. 在“预约情况”中可查看当天店面的整体预约情况。车主可在微信端新增预约信息。商家通过预约情况来提前安排店内工作。

  4. 证件拍摄如果使用？

    5. 商家通过该功能可拍摄车主的行驶证并上传，可自动将车主的行驶证信息转化为文字，主要用于后续对车主进行全方位服务使用。车主涉及违章、代缴年审、保险提醒等服务时间，均可在第一时间对车主提醒并代缴处理。

  5. 车主如果才能咨询我？

    6. 首先您必须要登录“车仆之家商家版”实时在线，这样车主在微信的公众号提问时可直接在线与您沟通交流，您只需及时回复车主的疑问即可。

  6. 我如何回复车主的咨询？

    7. 在“车主咨询”中会提示有车主发来的咨询消息，您只需要在线回复车主的疑问即可。当然您也可以将店面的一些优惠活动通过聊天告知给车主，增大车主来店消息的几率。
